首先是下面几个基础知识的复习

  1. virtual 用法
    • virtual 普通用法以及在destructor 中的作用
    • pure virtual 的作用
  2. extern关键字

1.virtual 用法

通过声明 Base类中的函数为 virtual 来使得 指针类型为 Base但是实际指向 derived的指针能过正确的调用派生类中的override了的函数，同时 通过声明基类的结构器
为virtual来让这种用法的类能够被正确的析构。
而pure virtual 则强制基类的被此声明修饰的函数被 override 掉，如果基类里的函数都加上这个关键字，那么他的效果就等同于接口了。

2. extern关键字

extern修饰的变量可以比如说在 a.h 声明文件里面 用extern var声明一下，然后在任何一个班汉这个头文件的.c文件里面定义一下，那么这个变量就对所有包含这个头文件的文件内容可见。

3.inline关键字

内联关键字，有两个作用：

  1. 编译器会尝试在调用该函数的地方直接替换掉内容，以此来节省stack 的空间
  2. 通常使用该关键字的函数大体比较简单，都会直接在头文件中直接定义，方便查看函数内容。
